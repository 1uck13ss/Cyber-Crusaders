Game Filtering Website

This is a project proposal for a game filtering website called Cyber Crusaders. The aim of this project is to provide gamers with a one-stop-shop to discover, research, and filter upcoming triple-A games based on their specific preferences.

Motivation
Gamers often face difficulty searching for upcoming triple-A games, as the process can be time-consuming and overwhelming. Existing sources lack a comprehensive and user-friendly platform for gamers to search and filter upcoming games according to their specific preferences. Thus, there is a need for a website that provides gamers with a one-stop-shop for discovering, researching, and filtering upcoming triple-A games to save time and enhance the gaming experience.

User Stories
The website aims to provide the following features to the users:

Search for upcoming triple-A games based on genre, platform, and release date.
Filter search results to show only games that match their specific interests and preferences.
View detailed information about upcoming triple-A games, trailers, and screenshots.
Save games to a personal wishlist or favorites list for future reference.
Features and Timeline
The website provides a quick and easy interface for gamers to view which popular games are upcoming. The website has limited features due to the limited time. The following features will be completed by the mid of June:

Set up the basic website infrastructure and design.
Develop the web scraper to scrape information about upcoming triple-A games.
Implement search and filtering options.
Additional features to be completed by the mid of July:

Providing recommendations with machine learning.
Tech Stack
The website will use the following tech stack:

Web Scraping Framework - Beautiful Soup
Relational Database Management System - MySQL
Web Development Framework - Django
Front-end Frameworks - React
Front-end tools - JavaScript, HTML, CSS
Machine Learning - SciKit
Overall, the website will provide a user-friendly interface and make searching and filtering upcoming triple-A games an easy task for the gamers.
